Приложение "CustomerBase" является иммитацией инструмента для работы с базой заказчиков, оно позволяет пользователям - вносить в базу новых заказчиков - просматривать профили существующих, оставлять о них отметки в профилях и вести переписку с другими пользователями приложения.
